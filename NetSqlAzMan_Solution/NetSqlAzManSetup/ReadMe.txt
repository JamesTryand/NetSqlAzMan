-----------------------------------------------
.NET Sql Authorization Manager Storage Database
-----------------------------------------------

IMPORTANT !!!
-------------
In order to use .NET Sql Authorization Manager, a NetSqlAzMan Storage DB must be created !


MS Sql Server 2000
------------------

1) Open Enterprise Manager.
2) Create a Database with a name like 'NetSqlAzManStorage'.
3) Open Query Analyzer and choose Database created above.
4) From Query Analyzer open and execute script named 'NetSqlAzMan_SqlServer.sql'.


MS Sql Server 2005
------------------

1) Open SQL Server Management Studio.
2) Create a Database with a name like 'NetSqlAzManStorage'.
3) Right click on Database created above and choose 'New Query'.
4) Load and execute script named 'NetSqlAzMan_SqlServer.sql'.
